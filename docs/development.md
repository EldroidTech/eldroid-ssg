# Development & Contribution

## Setting Up

```bash
git clone https://github.com/yourusername/eldroid-ssg.git
cd eldroid-ssg
cargo build
```

## Running Tests

```bash
cargo test
```

- Unit tests for core functionality
- Integration tests for build process
- Performance and security test cases

## Contributing
- Fork the repo and create a feature branch
- Submit a pull request with a clear description
- Follow Rust best practices and add tests for new features
